# Systematic-Review-Survival-Analysis-and-Granger-Causality-in-Demand-Forecasting-2020-2026-March-
This repository presents a comprehensive systematic literature review (SLR) and scientometric analysis exploring the intersection of survival analysis, causal inference (including Granger causality), and demand forecasting.

The project integrates a data-driven research pipeline combining automated bibliographic retrieval, relevance-based filtering, bibliometric analysis, network science, topic modeling, and statistical validation to map the intellectual and methodological landscape of the field.

# Objective
The ultimate goal of this project is to:

- Bridge the gap between survival analysis, causal inference, and demand forecasting
- Enable the development of next-generation predictive models
- Contribute to a unified causal-event forecasting framework

# Keywords
Survival analysis, causal inference, Granger causality, demand forecasting, customer churn, topic modeling, bibliometrics.

# Key Contributions

 1. Curated Dataset
- 1,879 collected publications
- 612 relevant papers (after filtering)
- Time span: 2020 – March 2026

2. Systematic Review (PRISMA-based)
Structured identification, screening, and eligibility process

3. Bibliometric & Scientometric Analysis
- Publication trends and venue distribution
- Author productivity and research fragmentation

4. Network Analysis
- Co-authorship network (Louvain community detection)
- Centrality measures (degree, betweenness, clustering)
- Identification of collaboration gaps

5.Semantic & Thematic Analysis
- Keyword co-occurrence networks
- Topic modeling using Latent Dirichlet Allocation (LDA)
- Identification of 5 major research themes

6. Method–Application Mapping
- Analysis of relationships between: (Survival models, Causal inference methods, Demand forecasting applications)
- Statistical validation using: (Fisher’s Exact Test & Pearson & Spearman correlations)
